# チケット016: デザイン再計画 - 背景融和性・枠形・フォント・色分け

## 概要

「デザインが芋っぽい」というフィードバックに基づき、以下の観点でデザインを根本的に改善する:
- 背景画像との融和性
- 枠の形状
- フォントの階層と使い分け
- イベント色分けの改善

参考: `Screenshots/スクリーンショット 2026-02-07 午後3.13.58.png`
（カラフルなカレンダーアプリ - 鮮やかな色分けで視認性が高い）

## 改善項目

### 1. 背景との融和性

**現状**: カード背景が不透明白、背景画像との調和がない

**改善案**:
- カード背景に半透明+ぼかし（ガラスモーフィズム）を適用
- テーマごとに背景との融和度を調整
  - simple: 85%不透明白（背景がうっすら見える程度）
  - modern: 70%不透明 + 背景ぼかし（ガラス効果）
  - dark: 70%不透明ダーク + 背景ぼかし

### 2. 枠の形状

**現状**: 角丸0〜15pxのみ、全テーマ同一パターン

**改善案**:
- テーマごとに特徴的な枠形状
  - simple: 角丸0px、細い枠線（1px）
  - modern: 角丸20px、枠線なし、影で浮遊感
  - pastel: 角丸12px、2px枠線、やわらかい色
  - dark: 角丸8px、ゴールド1px枠線
  - vibrant: 角丸16px、グラデーション枠（将来対応）

### 3. フォントの階層

**現状**: ヒラギノ角ゴシック W3 のみ、サイズ差も小さい

**改善案**:
- Bold/Regular の使い分け
  - 時刻: Bold（W6）、サイズ大
  - タイトル: Regular（W3）、サイズ中
  - 場所: Regular（W3）、サイズ小、薄い色
- macOS: `/System/Library/Fonts/ヒラギノ角ゴシック W6.ttc` をBold用に追加
- `FONT_PATHS` を `FONT_PATHS_BOLD` / `FONT_PATHS_REGULAR` に分割

### 4. イベント色分けの改善

**現状**: `DEFAULT_EVENT_COLORS` の彩度が低い、全テーマ共通

**改善案**:
- 彩度を上げて参考スクリーンショットに近づける
- テーマごとにイベントカラーパレットを設定（pastelは既に対応済み、他テーマにも拡張）
- 将来的にはカレンダーごとに色が変わる（チケット017と連携）
