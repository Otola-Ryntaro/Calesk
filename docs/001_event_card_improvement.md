# 001: 予定カード表示改善

## 優先度
**HIGH**

## 現状の問題
- 予定が1件しか表示できていない
- フォントサイズが大きすぎる
- 2件目以降の予定が表示されない

## 要件
- [x] 予定ごとに1カードで上から表示
- [x] 3件まで表示（それ以上は省略）
- [x] フォントサイズを適切に調整
- [x] カード内に複数予定を縦に並べる
- [x] 4件目以降は「+N件」として表示

## 技術仕様
- カード高さ: 150px → 200px（複数予定対応）
- 各予定の表示要素: 時刻、タイトル、場所、カラーバー
- フォントサイズ:
  - 日付ラベル: 18px → 16px ✅
  - 時刻: 24px → 16px ✅
  - タイトル: 20px → 14px ✅
  - 場所: 16px → 12px ✅

## 影響範囲
- `src/image_generator.py`: `_draw_single_card()` メソッド ✅
- `src/config.py`: フォントサイズ設定 ✅

## テスト観点
- [x] 予定0件の場合: 「予定なし」表示
- [x] 予定1件の場合: 1件のみ表示
- [x] 予定2-3件の場合: すべて表示
- [x] 予定4件以上の場合: 3件+「+N件」表示
- [x] フォントサイズが読みやすい
