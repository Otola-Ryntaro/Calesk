# 006: テーマシステム改善

## 優先度
**HIGH**

## 概要
背景画像を流用し、UI要素（カード、テキスト、グリッド）のスタイルのみ変更するテーマシステムに改善

## 問題点
- [x] 現在の'fancy'テーマはグラデーション背景で画像を置き換えている
- [x] 現在の'stylish'テーマは単色背景で画像を使わない
- [x] 背景画像を流用してUI要素のスタイルだけ変えるべき

## 要件

### 基本方針
- [x] すべてのテーマで背景画像を流用
- [x] UI要素（カード、テキスト、グリッド）のスタイルのみ変更
- [x] カードの影はオン・オフ可能

### 新しいテーマ設計

#### 1. simple（シンプル）- 既存維持
- [x] カード: 白背景（不透明）、角丸なし、薄いグレーの枠線
- [x] テキスト: 黒
- [x] 印象: クリーン、ミニマル

#### 2. modern（モダン）
- [x] カード: 半透明の白背景（85%）、大きめの角丸（15px）、影付き
- [x] テキスト: ダークグレー（#333）
- [x] 印象: 洗練された、Apple風

#### 3. pastel（パステル）
- [x] カード: 淡いパステルカラー（ピンク/ブルー/グリーン系）、角丸あり（10px）
- [x] テキスト: ダークグレー
- [x] 印象: 優しい、柔らかい

#### 4. dark（ダーク）
- [x] カード: 半透明の黒背景（70%）、小さい角丸（5px）、ゴールドの枠線
- [x] テキスト: 白〜薄いグレー
- [x] アクセント: ゴールド（#FFD700）
- [x] 印象: 高級感、夜間向き

#### 5. vibrant（鮮やか）
- [x] カード: 鮮やかな色の背景（ブルー/グリーン/オレンジ系）、角丸あり（12px）
- [x] テキスト: 白
- [x] 印象: 元気、ポップ

### 追加機能
- [x] カードの影：オン・オフ可能（設定で切り替え）

## 実装ファイル

### 1. src/themes.py（修正）
- [x] 'fancy'と'stylish'テーマを削除
- [x] 'modern', 'pastel', 'dark', 'vibrant'テーマを追加
- [x] `background_color`と`background_gradient`を削除
- [x] カードの透明度、角丸、影の設定を追加

### 2. src/config.py（修正）
- [x] デフォルトテーマを'simple'のまま維持
- [x] カードの影のオン・オフ設定を追加

### 3. src/image_generator.py（修正）
- [x] グラデーション背景生成コードを削除
- [x] カードの影を描画する機能を追加
- [x] 半透明カードの描画対応

### 4. tests/test_themes.py（修正）
- [x] 新しいテーマのテストを追加
- [x] 影のオン・オフテストを追加

## テスト項目
- [x] simple: 背景画像使用、白カード、角丸なし
- [x] modern: 背景画像使用、半透明白カード、角丸15px、影あり
- [x] pastel: 背景画像使用、パステルカード、角丸10px
- [x] dark: 背景画像使用、半透明黒カード、角丸5px、ゴールド枠線
- [x] vibrant: 背景画像使用、鮮やかなカード、角丸12px
- [x] カードの影：オン・オフで正しく動作

## 成功基準
- [x] すべてのテーマで背景画像が使用される
- [x] カードのスタイル（色、透明度、角丸）が正しく適用される
- [x] カードの影がオン・オフできる
- [x] テストが全て合格する
