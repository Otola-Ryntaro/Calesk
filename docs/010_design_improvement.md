# チケット010: デザイン改善

## 概要

「デザインが芋っぽい」というフィードバックを受けて、カレンダー壁紙アプリのデザインを洗練させる。

## 現在のデザイン

### 上段: 今日の予定カード（1カード1予定、最大3件）
- 時刻、タイトル、場所、カラーバーを表示
- 終了した予定はグレーアウト＋「終了」バッジ

### 下段: 週間カレンダー
- 月曜〜日曜、9:00-21:00
- イベントをカラーブロックで表示

### 5つのテーマ
1. **simple**: 白いカード、影なし、シンプル
2. **modern**: 85%透明カード、15px角丸、影付き
3. **pastel**: ピンク系カード、パステルカラー
4. **dark**: ダークグレーカード、金色の枠
5. **vibrant**: 鮮やかな青カード、白文字

## 問題点

1. **タイポグラフィの弱さ**: フォントサイズと太さの階層が不明瞭
2. **余白の不均等**: カード内のパディングが詰まりすぎ
3. **色使いの平坦さ**: 基本色で洗練度不足
4. **視覚的メリハリの欠如**: 重要情報の区別がつきにくい
5. **デザイン要素の古さ**: 単色のカラーバー、単純な角丸

## 改善案

### Phase 1: 即効性のある改善（優先度：HIGH）

#### 1. カラースキームの洗練化

**ファイル**: `src/themes.py`

- **minimal** (simple→改名): iOS風のオフホワイト、チャコール、薄い影
- **modern**: より透明に（alpha=230）、枠線なし、iOS風のApple Blue
- **pastel**: より上品な色（マットなラベンダー、ミントグリーン）
- **dark**: iOS風のダークグレー、ゴールドアクセント
- **vibrant**: 深いブルー、グラデーション対応

#### 2. タイポグラフィの改善

**ファイル**: `src/config.py`

FONT_SIZE_CARD_TIME = 28      # 現在: 18 → 大きく目立たせる

### Phase 2: レイアウト改善（優先度：MEDIUM）

#### 1. カードレイアウトの再構築

**ファイル**: `src/image_generator.py`

- 時刻を左上に大きく配置（Bold）
- タイトルを時刻の下に配置
- カラーバーを上部横バーに変更（高さ4px）または左端を極細に（2px）

#### 2. 「終了」バッジの改善

- 小さな丸いピル形状
- 薄いグレーと白文字
- サイズを70%程度に縮小

### Phase 3: 高度な視覚効果（優先度：LOW）

#### 1. グラデーション実装（vibrantテーマ）

**ファイル**: `src/image_generator.py`

```python
def _draw_gradient_card(self, draw, x, y, width, height, colors):
    """縦方向のグラデーション"""
    for i in range(height):
        ratio = i / height
        r = int(colors[0][0] * (1-ratio) + colors[1][0] * ratio)
        g = int(colors[0][1] * (1-ratio) + colors[1][1] * ratio)
        b = int(colors[0][2] * (1-ratio) + colors[1][2] * ratio)
        draw.line([(x, y+i), (x+width, y+i)], fill=(r, g, b))
```

#### 2. 影のぼかし効果

- 柔らかい影（Pillow制限内で）
- テーマごとに異なる影の色と強さ

#### 3. アイコン追加

- 時計アイコン（時刻の横）
- 場所アイコン（📍）
- Unicode記号を使用
